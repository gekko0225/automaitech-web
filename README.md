# Automaitech

Landing de captación y agendamiento para centros de estética, belleza y clínicas, construida con Astro.

## Ejecutar en Windows

```powershell
cd C:\Users\autom\Documents\Codex\2026-07-13\tengo\automaitech
npm install
npm run dev
```

Abre `http://localhost:4321`.

## Antes de publicar

- Confirmar el correo de contacto del pie de página.
- Confirmar los enlaces definitivos de Calendly y WhatsApp.
- Completar la política de privacidad.
- Sustituir ejemplos por casos reales cuando existan resultados documentados.
